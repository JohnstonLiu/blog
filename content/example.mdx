## Code Blocks

Here's a simple JavaScript function:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

// Example usage
console.log(greet("Johnston"));
```

A TypeScript interface example:

```typescript
interface User {
  id: number;
  name: string;
  email: string;
  isActive: boolean;
}

const user: User = {
  id: 1,
  name: "Johnston",
  email: "johnston@example.com",
  isActive: true
};
```

Some Python code:

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# Print first 10 Fibonacci numbers
for i in range(10):
    print(fibonacci(i))
```
